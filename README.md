# Proyecto QA y Automatización
proyecto creado por Yolanda Bernal
Repositorio creado para practicar y documentar:
- Testing manual
- Automatización de pruebas
- Validación de APIs
- Evidencias y reportes QA

---

## Tecnologías y herramientas

- Selenium
- Cypress
- Postman
- JavaScript
- Git y GitHub

---

## Estructura del proyecto

```plaintext
manual-testing/
automation/
api-testing/
evidencias/
reports/
```

---

## Contenido

### Manual Testing
- Casos de prueba
- Reportes de bugs
- Checklists

### Automation
- Scripts automatizados
- Pruebas funcionales
- Validaciones UI

### API Testing
- Collections Postman
- Requests HTTP
- Validación de respuestas

---

## Objetivo

Fortalecer conocimientos en QA y automatización mediante ejercicios prácticos y proyectos de prueba.

## Página web

Se creó un gestor de tareas desarrollado con HTML, CSS y JavaScript que permite:
- Crear tareas
- Visualizar tareas
- Marcar tareas como completadas
- Eliminar tareas
